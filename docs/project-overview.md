# Project Overview

## Purpose
The Number Guessing Game is a simple console-based application designed to demonstrate basic Python programming concepts including random number generation, user input handling, conditional logic, and loops.

## Target Audience
This project is suitable for beginners learning Python programming concepts. It can also be used as a fun educational tool for children learning about numbers and logical thinking.

## Key Features
- Random number generation between 1 and 100
- Input validation to ensure valid numbers
- Limited attempts (10) to make the game challenging
- Hints if the guess is too high or too low
- Option to play multiple rounds
- Clear and user-friendly interface

## Technical Details
- Language: Python 3.x
- Dependencies: None (uses only Python standard library)
- Platform: Cross-platform (works on Windows, macOS, Linux)
